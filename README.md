# movieOrderSystem

##	評分說明 （合計 100 分）
1. **前端主業部分**
<br>有分頁功能，點選定票連接到訂購票頁面 。

2. **前台訂票頁面**
<br>連動式選單，設有時間規定:只能訂購今天之後，場次有依照現實作。

3. **前台畫位頁面**
<br>check表單結合jquery做控制，可送資料到資料庫。

4. **後台電影列表**
<br>從資料庫撈取資料以列表顯示，富有改變順序，刪除等功能。

1. **後台訂單管理 / 10 分**
<br>有快速刪除功能，可依照所需做群組刪除。
